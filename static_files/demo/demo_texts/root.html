Welcome to the RESTx demo: The server's home page

RESTful API

RESTx provides a fully RESTful API. Everything the server has to offer can be explored simply by following links. The server's root URI is the only URI a client needs to know. Since we just accessed the root URI you can see links to other parts of the RESTx server, which we will explore shortly. To find out more about RESTx's RESTful API, please click here (opens in new window).

Content negotiation

RESTx supports content negotiation: The same information can be returned in different representations, depending on the client's preferences. Right now, RESTx has recognized that it is accessed via browser, this all information is rendered in useful HTML.

However, if you specify Accept: application/json as one of the HTTP request headers you see the same information rendered in plain JSON. To see and example of this, please click here (opens in new window).
