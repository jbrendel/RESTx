What is a component?

A component is a piece of stand-alone software, which performs a very specific function, such as accessing a database, a cloud service, a proprietary in-house API, etc. Components can also take advantage of RESTx and implement even complex data integration logic in just a few lines of code.

Serious data integration work often requires custom code at some point. That is why RESTx puts a lot of emphasis on making the writing of custom components as quick and easy as possible. You even have a choice of languages for your own components: You can write them either in Java, Python or server-side JavaScript.

The component API is very small and easy to learn. RESTx comes with command line utilities that create a ready-to-use component template in the language of your choice. You just need to fill in the blanks. To see how easy it is to start writing your own components, click here (opens in new window).
